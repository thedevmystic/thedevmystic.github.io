## Features
1. **React:** Modern component-based framework, for best UI/UX.
2. **HTML5:** Latest HTML5 compliant, for future scalability.
3. **TailwindCSS:** Utility-first styling, for responsive website.
4. **TypeScript:** Type-safe and modern JavaScript superset.
5. **Semantic Desgin:** Modular design for SEO, and clean readablity.
